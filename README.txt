Llenguatge: Java - versio: 10
IDE: IntelliJ IDEA
Execució: Classe Main -> Run 'Main.main()'
	Casos per executar:
		-Argument -> null: es desplegara una serie de menus
		-Argument -> !null: s'haurà d'inserir el nom del mètode 
				    MergeSort, QuickSort, SelectionSort o RadixSort; 
				    només un per execució, seguit pel o pels nombres 
				    que indiquen la funcionalitat.
